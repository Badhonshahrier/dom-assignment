Question-1: What are the different ways to select an element in the DOM?

When working with JavaScript and the Document Object Model (DOM),several ways to select elements 1.By ID:document.getElementById("id")=>Selects a single element with the specified id. 2.By Class Name:document.getElementsByClassName("class")=>select elements by their class attribute. 3.By Tag Name:document.getElementsByTagName()=>select elements by their tag name ( div, p, a). 4.By CSS Selector:document.querySelector() to select the first element that matches a CSS selector. document.querySelectorAll() to select all elements that match a CSS selector.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML sets or gets HTML content, rendering tags. innerText retrieves only visible text, ignoring hidden elements. textContent gets all text, including hidden content, without parsing HTML. innerHTML risks XSS attacks, while textContent is fastest. Use innerText for visible text, textContent for raw text, and innerHTML for HTML content.
Question-3: What is event delegation in the DOM?
Event delegation is a technique in the DOM where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, a single listener is placed on a common ancestor, improving efficiency. When an event occurs, it propagates up the DOM tree, allowing the parent to detect and handle it. This is useful for dynamically added elements and optimizing performance in large applications.
Question-4: What is event bubbling in the DOM?
Event bubbling is a DOM mechanism where an event triggered on a child element propagates up to its ancestors. When an event occurs, it first executes on the target element and then bubbles up through parent elements until reaching the root (document). This allows parent elements to handle events triggered by their children. Bubbling is useful for event delegation but can sometimes lead to unintended behavior, which can be prevented using event.stopPropagation(). Most DOM events, like click, support bubbling by default.
Question-5: How do you create, add, and remove elements using JavaScript?
To create elements, use document.createElement('tag'). Add them with parentElement.appendChild(newElement) or parentElement.insertBefore(newElement, referenceElement). Remove elements via parentElement.removeChild(elementToRemove) or elementToRemove.remove(). These methods allow dynamic DOM manipulation, enabling you to construct interactive web pages by adding or removing content as needed, directly through JavaScript.
